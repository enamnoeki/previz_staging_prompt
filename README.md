# Previz Brief Template

A structured template system for writing cinematic previsualization briefs — the kind of staging documents an art director hands to an image AI, a downstream illustrator, or a physical maquette photographer.

Briefs follow a 22-section format covering scene purpose, floorplan, asset descriptions, camera, lighting, composition, and AI prompt synthesis. The format is **floorplan-first**: the tabletop stage is built before the camera enters it, and before the puppets are described.

This repo is meant to be **forked or cloned per project**. Each project gets its own copy with a project-specific chassis, asset library, and brief collection.

---

## What this is for

Previz briefs are not finished illustrations. They are tabletop-maquette-style reference images — visibly handmade staging photographs that capture composition, lighting, blocking, and spatial logic. A downstream artist or AI pass translates the previz into the final illustration later.

Use this template when you need:

- Consistent composition language across a series (a tarot deck, a comic, a poster set, an animation pitch)
- A handoff document that survives translation between people (art director → illustrator) or tools (brief → image AI)
- A spatial logic that holds together across many scenes sharing recurring characters and props
- Version control and review on creative direction decisions

## What this is not for

- Finished illustration prompts
- Single one-off images where structure is overhead
- Photography direction for real shoots (some overlap, but this is built for fabricated maquette aesthetics)
